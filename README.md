# KidsBooks — UI/UX Design Mockup (Demo)

Interactive design mockup for the KidsBooks multilingual children's bookshop
mobile app (Mongolian market · QPay payments · delivery · annual membership).

**Live demo:** https://2026pro.github.io/kidsbooks-demo/

- 9 screens: Home, Search, Book detail, Cart, QPay checkout, Order tracking,
  Kids profiles, E-reader, Language picker
- 7-language UI switcher: Монгол · English · Français · Español · 한국어 · 中文 · Русский

*Design prototype only — no real data or payments.*
